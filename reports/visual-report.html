Формулы расчета

Коротко: какие математические формулы использовались для корреляций, вероятности победы и оценки качества.

Коэффициент корреляции Пирсона:
r = Σ((xᵢ - x̄)(yᵢ - ȳ)) / √(Σ(xᵢ - x̄)² Σ(yᵢ - ȳ)²)
Логистическая регрессия (вероятность победы хозяев):
p(homeWin=1|x) = 1 / (1 + e^(-(β₀ + β₁x₁ + ... + βₖxₖ)))
Пуассоновская модель голов:
λ(x) = exp(β₀ + β₁x₁ + ... + βₖxₖ), P(Y=y) = e^-λ * λʸ / y!
Метрики:
Accuracy = (TP+TN)/N, LogLoss = -(1/N)Σ[y log(p)+(1-y) log(1-p)], RMSE = √((1/N)Σ(y-ŷ)²)

Тепловая карта корреляций

Показывает линейную связь факторов между собой и с исходом: ближе к +1/-1 — сильнее связь, ближе к 0 — слабее.

Распределение голов

Гистограмма фактических голов Динамо и соперников. Нужна для проверки типичных диапазонов счёта.

Распределение разницы голов

Разница шайб (Динамо - соперник): помогает понять баланс матчей и частоту уверенных побед/поражений.

Динамика формы команд (средние очки за 5 матчей)

Rolling-форма по последним матчам. Используется как один из предикторов текущей силы команды.

Динамо Минск: ожидаемые и реальные голы

Сравнение прогноза Пуассона и реального числа голов в матчах. Наглядная проверка точности модели голов.

Метрики моделей

Accuracy/LogLoss — качество прогноза исхода, MAE/RMSE — ошибка прогноза количества голов.
